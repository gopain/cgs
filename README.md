# Create GraphQL Server


## Install
 
```bash
$ brew install yarn
$ npm i -g cgs
```

## Usage

### Init Project
```bash
$ cgs init <project-directory>
```
### Add Type
```bash
$ cgs add <path-to-schema.graphql>
```

## Develop

```bash
$ git clone https://github.com/wmzhai/cgs.git && cd cgs
$ yarn install
$ npm link
```
